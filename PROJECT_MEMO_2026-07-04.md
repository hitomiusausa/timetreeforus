# TimeTree For Us メモ

作成日: 2026-07-04

家族で使うための予定共有Webアプリ。TimeTreeの代わりとして、スマホでもPCでも家族の予定を見やすく共有できることを目指している。

## いまの公開状態

- 公開URL: `https://timetreeforus.vercel.app`
- コード管理: GitHub private repository
- 公開環境: Vercel Hobby
- データベース: Supabase Postgres
- GitHubの`main`ブランチにpushすると、Vercelへ自動デプロイされる。

## データの保存場所

入力した予定や家族カレンダーの情報は、Supabase Postgresに保存される。

GitHubにはコードだけが保存され、予定データは保存されない。

保存される主なデータ:

- ユーザー
- ログインセッション
- 家族カレンダー
- 家族メンバー
- 招待コード
- 予定
- 担当者
- タイトル候補

- 背景画像固定表示
- ロゴ / アイコン表示

- 開始日 / 終了日
- 繰り返し

  - 毎日

  - 複数選択
  - 全員

- 自由記述タイトルのラベル色選択
- 月表示内の予定ラベル表示

### 家族カレンダー

- 家族カレンダー作成
- 複数カレンダー作成
- カレンダー切り替え
- カレンダー名変更
- カレンダー削除
- 家族メンバー表示
- 招待コード表示
- 招待コードコピー
- メッセージで招待送信
- 招待リンクから参加
- 招待リンク経由の登録 / ログイン後の自動参加

招待メッセージの形式:

```text
TimeTree For Usで家族のカレンダーを作ろう。XXさんから「カレンダー名」カレンダーに招待されています。リンクをクリックしてさっそく参加してください。

参加リンク:
https://...
```

`XXさん` には、招待メッセージを送る人の登録名が入る。

### 書き出し

月ごとの予定をリスト形式で書き出せる。

- PDF
- Excel
- テキスト

PDFは1ヶ月分が1ページに収まる横向きレイアウト。タイトル欄の右端に書き出し日を表示する。

## PWA化

Safariなどで公開URLを開き、「ホーム画面に追加」するとアプリのように起動できる。

PWAで追加したもの:

- Web App Manifest
- iOS向けApple Web App設定
- service worker
- PWA用アイコン
- 静的画像とアイコンの軽いキャッシュ
- オフライン時の案内ページ

URLはPWA化前と同じ。

## セキュリティメモ

現在できていること:

- 所属しているカレンダーだけ表示
- URLに他のカレンダーIDを入れても、自分の所属カレンダーからのみ選択
- 予定追加 / 編集 / 削除時にカレンダー所属チェック
- 書き出しAPIでもカレンダー所属チェック
- カレンダー名変更 / 削除は管理者のみ
- パスワードはbcryptでハッシュ化
- DB接続情報は`.env`とVercel環境変数で管理

家族内で使う前提では最低限の安全性は確保している。周りの家族に使ってもらいながら、必要に応じて追加対策を入れる。

今後追加すると安心なもの:

- ログイン試行制限
- 招待コードの再発行 / 無効化
- パスワード再設定
- セキュリティヘッダー強化
- 利用規約 / プライバシーポリシー
